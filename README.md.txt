Platform: Windows
IDE: Dev-C++

1. Project Overview

This project implements a Digital Attendance System for hour-based lectures. It replaces traditional paper-based attendance with a console-based C++ application, allowing:

Student registration

Lecture session creation

Attendance marking (Present, Absent, Late)

Attendance reports and summaries

Data is stored in text files for persistence between program runs.

2. Features
Week 1 – Student Management

Register students with index number and full name

View all registered students

Week 2 – Lecture Session Management

Create a lecture session with course code, date, start time, and duration

Attendance file is automatically generated for the session

Week 3 – Attendance Marking

Mark students as Present (P), Late (L), or Absent (A)

Updates attendance file safely

Week 4 – Reports and Summary

Display attendance report for any session

Count total Present, Absent, and Late

Calculate attendance percentage
